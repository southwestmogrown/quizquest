# What is Go?

Go is a statically typed, compiled programming language designed at Google.

It emphasizes simplicity, reliability, and efficiency — making it a popular
choice for backend services, CLIs, and distributed systems.

## Key Features

- Fast compilation
- Garbage collected
- Built-in concurrency via goroutines and channels
- Strong standard library
