# What is Go?

Go (also called Golang) is a statically typed, compiled programming language designed at Google. It was first released in 2009 and has become one of the most popular languages for building scalable backend systems, CLI tools, and cloud-native infrastructure.

## Why Go?

- **Simple syntax** — Go has a small number of keywords, making it fast to learn.
- **Compiled** — Go compiles directly to machine code, giving it performance comparable to C.
- **Built-in concurrency** — Goroutines and channels make writing concurrent programs straightforward.
- **Strong standard library** — HTTP servers, JSON handling, testing, and more are built in.
- **Fast compile times** — Even large codebases compile in seconds.
